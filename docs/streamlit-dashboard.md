# Streamlit Dashboard Plan

Streamlit is the primary analytics layer. The app should answer concrete questions about the
Stack Overflow Developer Survey instead of mirroring raw tables.

## Launch

```powershell
uv run streamlit run app/streamlit_app.py
```

## Data Source

The dashboard reads generated pipeline outputs:

- `data/processed/Cleaned_SurveyData.xlsx`
- `data/processed/Cleaned_SurveyData.manifest.json`
- `data/processed/multiyear/core_survey.csv`
- `data/processed/multiyear/technology_counts.csv`

These files are not committed. They are regenerated by:

```powershell
uv run python -m stackoverflow_analytics.pipeline
uv run python -m stackoverflow_analytics.multiyear_eda
```

## Dashboard Structure

### Overview

Questions:

- How many respondents are represented after filtering?
- Which countries dominate the dataset?
- How is the sample distributed by education, age, remote work, and developer type?

Implemented visuals:

- Respondents, countries, median salary, and top language KPI cards.
- Respondents by country.
- Remote work distribution.
- Education distribution.
- Age distribution.

### Salary

Questions:

- Which countries have the highest median salary?
- How does salary vary by experience and education?
- How skewed is the salary distribution?

Implemented visuals:

- Median salary by country.
- Median salary by professional experience.
- Median salary by education.
- Salary distribution histogram.

### Technologies

Questions:

- Which languages are most used?
- Which technologies are most desired but less used?
- How broad are respondent technology stacks?

Implemented visuals:

- Language popularity.
- Worked-with vs wanted language comparison.
- Technology popularity by category.
- Technology stack breadth.

### AI Adoption

Questions:

- How many respondents use or plan to use AI tools?
- What is the sentiment toward AI?
- Which respondent groups are more positive or skeptical?

Implemented visuals:

- AI selection distribution.
- AI sentiment distribution.

### Work And Career

Questions:

- How is remote work distributed across countries and developer types?
- How does job satisfaction vary by remote mode and experience?
- Which industries are most represented?

Implemented visuals:

- Developer type distribution.
- Job satisfaction distribution.
- Remote work by country table.

### Trends

Questions:

- How did median salary change across survey years?
- How did remote, hybrid, and in-person work change?
- How did AI adoption change from the AI-era surveys onward?
- Which worked-with languages dominate across years?

Implemented visuals:

- Median salary by year.
- AI adoption by year.
- Remote work by year.
- Top worked-with languages by year.

## Implementation Notes

- Prefer reusable transformation functions in `stackoverflow_analytics/dashboard_data.py`.
- Keep Streamlit components thin: layout, filters, and visual rendering only.
- Move from Excel to Parquet when dashboard startup time becomes a blocker.
- Avoid committing generated files under `data/processed/`.
